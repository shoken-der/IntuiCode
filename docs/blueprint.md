# **App Name**: IntuiCode

## Core Features:

- DSA Problem Upload & Analysis: Users can upload problem statements (URL, paste, screenshot); AI automatically detects the core coding pattern, problem difficulty, tested concepts, and provides a concise explanation of 'why' the pattern fits, guiding initial thinking.
- AI Guided Intuition Development: An immersive chat interface where an AI mentor (orb/avatar) guides users step-by-step to build intuition, asking adaptive questions and providing subtle hints to optimize solutions without revealing the answer directly.
- Interactive Coding Workspace: A futuristic coding lab featuring a Monaco editor for real-time code writing in multiple languages (Java, C++, Python), integrated with dynamic AI hints accessible upon request to overcome roadblocks.
- AI Simulated Technical Interview (Viva): Users engage in a realistic viva where the AI interviewer asks targeted questions about their solution's complexity, edge cases, alternative approaches, and general CS fundamentals, preparing them for real-world interviews.
- Comprehensive AI Performance Report: Post-session, AI generates a detailed report evaluating technical proficiency, communication clarity, optimization thinking, identifying strengths, weaknesses, and a personalized roadmap for improvement, saved for user review.
- Secure User Authentication & Session Management: Provides secure user registration, login via JWT authentication, and session management to personalize the learning experience and track individual progress over time, utilizing PostgreSQL as the database.
- Dynamic User Dashboard & Progress Tracking: A 'futuristic AI command center' dashboard displaying animated interview readiness scores, floating stats cards for progress, streak tracking, recent session history, and identifying strong/weak topics, all retrieved from PostgreSQL.

## Style Guidelines:

- Primary accent color: A deep, intelligent indigo-blue (#2E2E6B). This hue is chosen to evoke a sense of calm sophistication and technical depth, ideal for a mentor-like AI experience.
- Background color: A nearly black, subtly desaturated blue-gray (#121216), providing a dark, matte, futuristic canvas that allows other elements to glow and stand out with premium depth.
- Accent and neon glow color: A vibrant, clear sky blue (#66DAFF). Positioned to the 'left' of the primary hue on the color wheel, its brightness and saturation provide a striking contrast, perfect for neon gradients, glowing elements, and interactive highlights, symbolizing clarity and advanced technology.
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-forward, and clean aesthetic, suitable for impactful titles and key UI elements.
- Body font: 'Inter' (sans-serif) for its highly legible and neutral design, ensuring comfortable reading of longer explanations and problem statements, aligning with a professional, objective interface.
- Code font: 'Source Code Pro' (monospace) for optimal readability and precise alignment within the coding workspace and for any displayed code snippets.
- Utilize sleek, minimalistic line icons that incorporate subtle neon glow effects, designed to complement the glassmorphism and animated components without adding visual clutter. Prefer geometric and abstracted symbols over literal representations.
- A sophisticated, multi-layered layout inspired by '3D glassmorphism' and 'floating cards.' Elements will exhibit premium depth shadows, subtle mesh gradients in backgrounds, and soft glow lighting, creating an immersive, futuristic 'AI command center' feel.
- Implement subtle but pervasive animations: floating animated particles in backgrounds, dynamic border effects on interactive elements, 'animated thinking indicators' for the AI mentor, smooth typing animations, and an 'animated interview readiness score ring' to provide fluid and responsive feedback.
